Editing with Claude

Ask Claude to change anything on your site using plain language — no code required.

The chat panel

The right side of every site window has a chat panel. Type what you want to change and Claude takes care of the rest. You can ask for anything from a wording tweak to a layout change, and Claude will figure out which files to edit.

Edits appear in your site

When Claude finishes, the changes are written to your files and the live preview updates automatically. An overlay highlights what changed directly in the preview so you can see the result at a glance.

Every edit is tracked

Each successful edit appears as a row in the chat showing the file that changed and how long ago. Each row has an Undo button so you can reverse any edit without leaving the app.
