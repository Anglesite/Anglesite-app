Accounts and setup

Connect your GitHub and Cloudflare accounts so Anglesite can back up and publish your site.

GitHub

Sign in to GitHub from within Anglesite to back up your site and publish it. Anglesite will prompt you to sign in when you first need it.

Cloudflare

When Anglesite needs to deploy to Cloudflare, it will prompt you to provide a Cloudflare API token. You can create one in your Cloudflare dashboard.

Stored securely

Your credentials are stored in the macOS Keychain — not in any file inside your site folder.
