Accounts and setup

Connect your GitHub and Cloudflare accounts so Anglesite can back up and publish your site.

GitHub

Your site's Source folder is an ordinary Git repository, so you can add a GitHub remote with the gh command-line tool or git remote in Terminal. Once the repository has an origin configured, Anglesite uses it to back up and publish your site.

To let Anglesite push to GitHub, add a GitHub personal access token in Settings ▸ Advanced. Create a fine-grained token with Contents: Read and write access for your site's repository at github.com/settings/tokens.

Cloudflare

When Anglesite needs to deploy to Cloudflare, it will prompt you to provide a Cloudflare API token. You can create one in your Cloudflare dashboard.

Stored securely

Your credentials are stored in the macOS Keychain — not in any file inside your site folder.
