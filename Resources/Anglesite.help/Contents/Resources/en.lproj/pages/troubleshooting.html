Troubleshooting

Quick answers to common problems.

The preview won't load

Give it a moment — the preview needs a little time to start up after you open a site. If it still doesn't appear, check the Debug pane for error messages. Closing and reopening the site window restarts the preview server.

My deploy is blocked

Open the readiness dot's popover (or read the sheet that appeared when the deploy was blocked) to see which checks failed. Fix those issues, then try deploying again. See Health and deploy readiness for more detail.

An edit didn't apply

Check the chat for a message explaining what went wrong. If an edit fails partway through, Anglesite restores the original file so nothing is lost.

Where are my files?

Your site is an ordinary folder on your Mac. You can open it in Finder at any time and work with the files directly in any app you like — Anglesite doesn't lock you in.
