Deploying your site

When your site is ready to share with the world, deploy it from the site window with a single action.

Open the Deploy drawer

Use the Deploy drawer in the site window to publish your site. Before anything is sent live, Anglesite runs its standard pre-deploy checks automatically.

When a check fails

If any pre-deploy check fails, the deploy is blocked and a sheet explains what needs to be fixed. Anglesite cannot skip these checks — it surfaces the problem so you can resolve it rather than letting a broken site go live. Fix the issue, then try deploying again.

Accounts you'll need

Deploying uses your connected GitHub and/or Cloudflare account. If you haven't set those up yet, see Accounts and setup.
