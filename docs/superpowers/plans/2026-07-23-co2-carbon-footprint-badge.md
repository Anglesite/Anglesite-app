# Carbon Footprint Badge (CO2.js) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `.co2Badge` Bucket-3 integration that shows an estimated per-page CO2 footprint, computed from each built page's byte weight via `@tgwf/co2`, recomputed on every `astro build` — no client-side JS shipped to visitors.

**Architecture:** Unlike #684's `greenHostCheck` (a one-time async check at wizard-setup time, baked into `.site-config`), this integration's computed value must be **fresh on every build** (per the issue: recomputed automatically, not a stale wizard-time snapshot). So there is no Swift-side async work at all — the Swift side is purely declarative (register a descriptor, ship a component + optional dedicated page, write one config key for placement). All the "smart" behavior lives in a new Astro integration (`Resources/Template/scripts/co2-badge.ts`), registered in `astro.config.ts` alongside the existing `redirects()` integration, whose `astro:build:done` hook fires after Astro has written every page to `dist/`. Since the page's own final byte size (including its own badge markup) isn't knowable until after Astro renders it, the `CO2Badge.astro` component renders a placeholder token (`CO2_PLACEHOLDER`) instead of a value; the post-build hook walks `dist/**/*.html`, measures each file's actual byte size, computes the CO2 estimate, and string-replaces the placeholder with the real number. This mirrors `redirects.ts`'s `astro:build:done`-writes-into-`dist/` shape structurally, but goes further (patching an already-rendered file rather than writing a brand-new one) — there is no existing precedent for that specific move in this template, confirmed by research.

**Scope note (resolved with the repo owner before this plan was written):** the original issue described **two** providers (`co2js` and `websitecarbon`, modeled like `donations`' provider array). Research found Website Carbon Calculator has no static badge product (only a JS widget, exactly what the issue said to avoid for ADR-0008), and as of July 2025 their underlying API restricts free access to registered non-profits/verified academics — the widget would show "No result" for most Anglesite owners regardless of consent-gating. The `websitecarbon` provider is dropped entirely; this integration ships **CO2.js only**, provider-less (matching `carbonTxt`/`greenHostCheck`'s shape: `providers: []`), not the two-provider `donations` shape the issue originally sketched. See the comment thread on issue #686 for the full research trail.

**Tech Stack:** Astro integration (TypeScript, Node `astro:build:done` hook), `@tgwf/co2` npm package (confirmed working: `new co2({model: "1byte"}).perByte(bytes, greenHosting?)` returns grams CO2e as a float — verified live in this environment, `perByte(100000)` → `0.0291`, `perByte(500000)` → `0.1454`, `perByte(100000, true)` → `0.0232` for green hosting), Swift `IntegrationDescriptor` framework, Swift Testing, Node's built-in `node:test` (this template's existing convention for `scripts/*.test.ts` — see `scripts/redirects.test.ts` — run via `npx tsx --test scripts/<file>.test.ts`, confirmed working in this environment without a prior `npm install` needed for `tsx` itself, though `@tgwf/co2` does need to be an installed dependency for `co2-badge.test.ts` to import it).

```typescript
// Resources/Template/scripts/co2-badge.ts (Task 1 portion — Task 2 appends to this same file)
import { co2 } from "@tgwf/co2";

/// Rendered by CO2Badge.astro instead of a real value: the page's own final byte size (which
/// includes this badge's own markup) isn't knowable until after Astro has finished rendering it,
/// so the component renders this token and a post-build step (see co2Badge() below) patches the
/// real estimate into the already-written HTML.
export const CO2_PLACEHOLDER = "__ANGLESITE_CO2_PENDING__";

const estimator = new co2({ model: "1byte" });

/// UTF-8 byte size of a page — the transfer-size proxy this badge estimates from. This measures
/// the page's own rendered HTML only, not its full asset weight (CSS/JS/images) — an honest,
/// documented approximation, not a claim of total page-weight accuracy.
export function byteLength(html: string): number {
  return Buffer.byteLength(html, "utf-8");
}

/// Grams of CO2e for one page view of `bytes`, via CO2.js's byte-based model.
/// `greenHosting` lowers the estimate when the host is on a verified green-energy provider.
export function estimateGramsPerByte(bytes: number, greenHosting?: boolean): number {
  return estimator.perByte(bytes, greenHosting);
}

/// Two-decimal display string, e.g. "0.35".
export function formatGrams(grams: number): string {
  return grams.toFixed(2);
}

/// Replaces every occurrence of CO2_PLACEHOLDER in `html` with the formatted grams value.
/// A no-op when the placeholder isn't present (the integration isn't installed on this page).
export function patchHtml(html: string, grams: number): string {
  if (!html.includes(CO2_PLACEHOLDER)) return html;
  return html.split(CO2_PLACEHOLDER).join(formatGrams(grams));
}
```

- [ ] **Step 1: Append the integration to `co2-badge.ts`**

Append to the end of `Resources/Template/scripts/co2-badge.ts` (after Task 1's exports):

```typescript
import { readdirSync, readFileSync, statSync, writeFileSync } from "node:fs";
import { extname, join } from "node:path";
import { fileURLToPath } from "node:url";
import type { AstroIntegration } from "astro";

/// Recursively collects every `.html` file under `dir`. Mirrors microformats.ts's walkHtml.
function walkHtmlFiles(dir: string): string[] {
  const out: string[] = [];
  let names: string[];
  try {
    names = readdirSync(dir);
  } catch {
    return out; // dir absent — not an error here
  }
  for (const name of names) {
    const full = join(dir, name);
    if (statSync(full).isDirectory()) out.push(...walkHtmlFiles(full));
    else if (extname(full) === ".html") out.push(full);
  }
  return out;
}

/// Patches every built page carrying CO2_PLACEHOLDER with a real, per-page CO2 estimate computed
/// from that page's own final byte size. A complete no-op for sites that haven't installed the
/// co2Badge integration (no page will contain the placeholder), matching redirects()'s
/// always-registered-but-conditionally-active shape.
export default function co2Badge(): AstroIntegration {
  return {
    name: "anglesite-co2-badge",
    hooks: {
      "astro:build:done": ({ dir }) => {
        const distDir = fileURLToPath(dir);
        for (const file of walkHtmlFiles(distDir)) {
          const html = readFileSync(file, "utf-8");
          if (!html.includes(CO2_PLACEHOLDER)) continue;
          const grams = estimateGramsPerByte(byteLength(html));
          writeFileSync(file, patchHtml(html, grams));
        }
      },
    },
  };
}
```
